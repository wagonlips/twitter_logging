wagonlips.com uses Bash, Python, iPython, LFTP, the Twitter API, github.com, and the Python libraries python-twitter, TextBlob, and pytz.
wagonlips.com is a kind of literary mashup, mostly random, a curious blending of classic poetry and prose, with doodles, much of which came from Project Gutenberg and was read with the help of genius.com.
This work is in the Public Domain CC0 1.0 Universal (CC0 1.0) Public Domain Dedication.
Any resemblance to anything meaningful might have been intentional but is probably coincidental. Not to be taken internally.
